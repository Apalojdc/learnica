Module 1: Introduction à React

Découvrez les bases de React et créez votre premier composant

1. Qu'est-ce que React ?

React est une bibliothèque JavaScript créée par Facebook pour construire des interfaces utilisateurs interactives. Ses principaux avantages sont :

  • Virtual DOM pour des performances optimales
  • Architecture basée sur les composants
  • Grande communauté et écosystème riche
  • Apprentissage progressif
Note importante : React n'est pas un framework complet comme Angular, mais une bibliothèque focalisée sur l'interface utilisateur.

2. Configuration de l'environnement

Pour commencer avec React, vous aurez besoin de :

Prérequis :

  • Node.js (version 14 ou supérieure)
  • npm (inclus avec Node.js)
  • Un éditeur de code (VS Code recommandé)

Création d'un projet React :

npx create-react-app mon-premier-projet
cd mon-premier-projet
npm start
Structure du projet :
  • src/ : Vos fichiers source
  • public/ : Fichiers statiques
  • package.json : Dépendances

3. Votre Premier Composant React

import React from 'react';

function MonPremierComposant() {
    return (
        <div>
            <h1>Bonjour React !</h1>
            <p>Ceci est mon premier composant</p>
        </div>
    );
}

export default MonPremierComposant;

Pour utiliser ce composant dans App.js :

import MonPremierComposant from './MonPremierComposant';

function App() {
    return (
        <div>
            <MonPremierComposant />
        </div>
    );
}
